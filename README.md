# VAST_presence_only_integration
Developing presence-only data integration methods with the vector autoregressive spatio-temporal (VAST) modelling platform. The Code folder gives R code needed to run the freshwater application for the paper "Integrating presence-only data into spatio-temporal models to predict taxon encounter probability and density in freshwater and marine environments". See https://github.com/ACharsley/VAST_VME_modelling for the code needed to run the marine application.

'Analyses' (containing the outputs of the analyses), 'Data_raw' (containing all raw data), and 'Data_processed' (containting all processed data) folders have not been uploaded to Github due to size limitations. However, these folders are available on request. Please email me at Anthony.Charsley@niwa.co.nz
